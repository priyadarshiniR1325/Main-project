# Employee Engagement & Workplace Satisfaction Analysis
 **Project Overview**

This project focuses on analyzing employee survey data to understand employee satisfaction, work-life balance, workplace experience, and wellbeing. The analysis aims to uncover meaningful patterns that can support HR decision-making and improve organizational performance.

 **Objectives**

To assess overall employee job satisfaction and engagement levels

To identify key drivers of workplace satisfaction and dissatisfaction

To analyze department-wise and demographic trends in employee wellbeing

To provide actionable business recommendations based on insights

To present findings using an interactive Power BI dashboard

 **Dataset Description**

Source: Employee Survey Dataset

Records: 3,025 employees

Variables: 23 features covering demographics, work environment, workload, wellness, and satisfaction

Data Quality: No missing values and no duplicate records

 **Methodology**

Data understanding and preparation

Exploratory data analysis using visualizations

Identification of trends, correlations, and anomalies

Dashboard creation using Power BI

Business interpretation and storytelling

 **Key Insights**

Work-life balance strongly influences job satisfaction

Certain departments show lower engagement levels

Long working hours negatively affect employee wellbeing

Positive workplace environments improve morale

Gender satisfaction differences are minimal

 **Business Recommendations**

Introduce flexible work policies

Improve workload distribution in low-performing departments

Strengthen leadership and employee support programs

Invest in training and development initiatives

Use dashboards to monitor workforce engagement

**Dashboard & Reporting**

An interactive Power BI dashboard was developed to visualize key workforce trends and support data-driven HR decisions. A detailed PDF report accompanies the dashboard, explaining insights and recommendations.

 **Conclusion**

This project demonstrates how employee survey data can be transformed into actionable insights that enhance workplace satisfaction, productivity, and retention through informed business decisions.
